¿Qué es la Programación Lineal?

La programación lineal es una técnica matemática poderosa utilizada para encontrar el valor máximo o mínimo de una función objetivo, sujeta a un conjunto de restricciones lineales.

Es ampliamente utilizada en áreas como economía, ingeniería, logística y gestión de recursos para tomar decisiones óptimas en situaciones complejas.

Función Objetivo:

Maximizar: Z = c₁x₁ + c₂x₂ + ... + cₙxₙ

Restricciones:

a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ ≤ b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ ≤ b₂
⋮
x₁, x₂, ..., xₙ ≥ 0
